# 🎬 CineTrack

Application **React Native** (Expo) de découverte et suivi de films avec **IA embarquée**.

## 📖 Documentation

La documentation interactive avec les maquettes est disponible sur **GitHub Pages** :

### 👉 [**Consulter les maquettes et le roadmap**](https://othila-academy.github.io/expo-start-project-cinetrack/)

## ✨ Fonctionnalités principales

- **Accueil** — Découvrez les films populaires
- **Recherche** — Trouvez rapidement un film
- **Favoris** — Gérez votre liste personnalisée
- **Détails** — Consultez synopsis, notes et informations
- **CineBot** — Obtenez des recommandations personnalisées par IA

## 🚀 Stack technologique

- **React Native 0.81.5** + **Expo 54**
- **Expo Router** pour la navigation
- **Context API** + **AsyncStorage** pour l'état
- **API IA** pour les recommandations

## 🛠️ Développement

### Installation

```bash
npm install
```

### Lancer l'app

```bash
npm start
```

Options :
- `npm run android` — Sur Android
- `npm run ios` — Sur iOS  
- `npm run web` — Sur navigateur

## 📅 Roadmap

Le développement suit un calendrier **3 jours** :

- **Jour 1** — Accueil, détails, favoris de base
- **Jour 2** — Recherche, persistance, UX raffinée
- **Jour 3** — CineBot, API IA, streaming

Voir la [**documentation complète**](./docs/README.md) pour plus de détails.
